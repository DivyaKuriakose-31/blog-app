# Nexus Blog Application

A modern, responsive blog application built with Next.js, featuring Server-Side Rendering (SSR), a custom proxy implementation, and dynamic article routing.

## 🚀 Tech Stack
*   **Framework**: Next.js (App Router)
*   **Styling**: Tailwind CSS
*   **Data Fetching**: React Query
*   **Language**: TypeScript

## ✨ Key Features
*   **Dynamic Routing**: Browse individual articles with dedicated pages.
*   **Server-Side Rendering**: Optimized performance and SEO.
*   **Search Functionality**: Easily filter and find content.
*   **Authentication UI**: Secure sign-in and sign-up interfaces.

## 🛠️ How to Run Locally
1. Clone this repository: `git clone https://github.com/DivyaKuriakose-31/blog-app.git`
2. Install dependencies: `npm install`
3. Start the development server: `npm run dev`
4. **Open**[http://localhost:3000].
5.**Live Demo**: [https://blog-app-blond-sigma.vercel.app/]
